Pour générer l'exe du programme Import_Agrigest

1-exporter le jar se mettre sur le projet, export, Runnable jar file
2- lancer Launch4J avec comme paramètre le fichier Import_agrigest.xml
3- lancer innoSetup à partir du fichier setupImportAgrigest.iss

récupérer le fichier import_agrigest.exe pour le donner si besoin sinon il est embarqué le webapp et est installable à partir de l'écran d'importAgrigest
