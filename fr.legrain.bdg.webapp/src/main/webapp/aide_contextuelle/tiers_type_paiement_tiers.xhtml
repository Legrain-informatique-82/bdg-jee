Type de Paiement

Définissez un type de paiement par défaut pour ce Tiers.

Lors de la saisie d'un document, comme par exemple une facture, le programme vous proposera ce type de paiement par défaut. Bien entendu, vous pourrez toujours le modifier lors de la saise dans votre facture.

Vous pouvez définir autant de Type de Paiement que vous le voulez dans la gestion des paramètres.
