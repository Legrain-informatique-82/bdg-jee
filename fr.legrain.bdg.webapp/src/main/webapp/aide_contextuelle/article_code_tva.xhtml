Code TVA

Associez le code TVA qui devra être utilisé par le programme pour le calcul du montant de TVA lorsque vous réaliserez une vente, ou toutes autres opérations, de l'article courant.

Les taux de TVA peuvent être modifiés par l'état et il incombe aux professionnels d'appliquer ces changements dans les délais légaux.

En savoir plus : Service Public

Vous pouvez définir les Codes TVA dont vous avez besoin dans la gestion des paramètres.
