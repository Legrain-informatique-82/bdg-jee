Type de Tarif

Définissez un type de tarif par défaut pour ce Tiers.

Lors de la saisie d'un document, comme par exemple une facture, le programme vous proposera ce type de tarif par défaut s'il est associé à un prix d'un article. Bien entendu, vous pourrez toujours le modifier lors de la saise dans votre facture.

Vous pouvez définir autant de Type de tarif que vous le voulez dans la gestion des paramètres.
