Articles - Gestion du catalogue

Saisissez ici tous vos articles à vendre de votre catalogue ainsi que les articles achetés utilisés dans vos fabrications et/ou réceptions.

Vous pouvez distinguer les articles en produits finis (ventes) et/ou en matière première (fabrications).

Vous pouvez aussi les regrouper par familles et associer une famille par défaut.

Définissez les unités U1 et U2, ainsi que l'unité de référence pour une meilleure visibilité dans vos stocks.

Si vous gérez les réceptions (menu stocks), vous avez la possibilité d'automatiser la constitution de la liste des fournisseurs qui vous fournissent un article, sinon vous pouvez le faire manuellement.

Si vous gérez les fabrications et/ou le suivi qualité, vous pouvez créer ici la liste des contrôles qualités ,ainsi que les recettes de fabrications de l'article courant.

Liaison Comptabilité : Indiquer le code comptable et le numéro de compte général. Ces informations sont reprises lors de l'exportation comptable.

Pour ajouter un article, cliquez sur le bouton de menu

Pour modifier une fiche, cliquez sur le bouton de menu

Pour accéder à la liste des articles du catalogue, cliquez sur le bouton de menu
