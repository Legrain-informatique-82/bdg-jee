Facture - Ecran de saisie et de consultation

Dans cet écran, vous pouvez créer, modifier, consulter ou supprimer vos factures.

Trois étapes pour la saisie de vos factures

Etape 1 - Informations d'entête.

Vous choisissez ici à quel client s'adresse la facture, la date de validité.

Le programme rempli pour vous toutes les informations qu'il lui est possible de récupérer depuis la fiche client, génère le code facture, et calcule les dates.

Pour passer à l'étape suivante, cliquez sur le bouton suivant en bas de l'écran.

Etape 2 - Les lignes de votre facture.

Dans la grille de saisie, choisissez les articles qui constituent votre facture. Le programme pré-rempli toutes les informations issues de la fiche article. Chaque valeur est modifiable en fonction de vos besoins.

Pour saisir un commentaire, il vous suffit de ne rien saisir dans la colonne article.

Pour passer à l'étape suivante, cliquez sur le bouton suivant en bas de l'écran.

Etape 3 - Récapitulatif de votre facture.

Dans cette dernière étape, vous pouvez saisir une remise globale et voir les différents totaux et ventilation TVA.

Pour enregistrer la facture, cliquez sur le bouton enregistrer situé dans la barre d'icone en haut de l'écran..
