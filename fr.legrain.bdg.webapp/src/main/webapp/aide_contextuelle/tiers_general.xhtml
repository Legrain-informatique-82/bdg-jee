Tiers - Gestion des contacts

Saisissez ici tous les contacts que vous voulez enregistrer dans votre gestion des contacts.

Afin des les différencier, vous pouvez leur affecter un type (clients, fournisseurs, salariés, etc ....)

Vous pouvez aussi les regrouper par familles, indiquer la localisation TVA, le type de saisie (par exemple TTC pour des particuliers).

Liaison Comptabilité : Indiquer le code comptable et le numéro de compte général. Ces informations sont reprises lors de l'exportation comptable.

Pour ajouter un tiers, cliquez sur le bouton de menu

Pour modifier une fiche, cliquez sur le bouton de menu

Pour accéder à la liste des tiers, cliquez sur le bouton de menu
