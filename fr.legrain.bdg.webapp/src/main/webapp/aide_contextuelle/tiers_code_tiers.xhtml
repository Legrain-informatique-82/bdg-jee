Code Tiers

Le code de tiers est un code unique qui permet de retrouver facilement chaque tiers dans tout le programme

Ce code est généré automatiquement par le programme. Vous pouvez en saisir un, mais à condition qu'il soit unique.

Seul les administrateurs de votre dossier sont autorisés à modifier les règles de génération de codes.
