Bureau de gestion

Documentation basée sur OpenAPI/Swagger
Le dossier Bureau de gestion cible doit être passé dans tous les requêtes HTTP en utilisant le paramètre X-Dossier.
La méthode d'authentification recommandée est celle utilisant la méthode des Bearer Token de type JSON Web Token (JWT) et l'entête HTTP standard 'Authorization' (rfc6750).
